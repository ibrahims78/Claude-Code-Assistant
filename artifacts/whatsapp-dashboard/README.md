# WhatsApp Manager

لوحة تحكم لإدارة جلسات واتساب متعددة عبر wppconnect.

**المسار:** `/whatsapp/` | **المنفذ:** 23097

## الصفحات

| الصفحة | الرابط | الوصف |
|-------|--------|-------|
| لوحة التحكم | `/` | إحصائيات عامة |
| الجلسات | `/sessions` | قائمة الجلسات |
| تفاصيل جلسة | `/sessions/:id` | 5 تبويبات: معلومات، رسائل، QR، webhook، ميزات |
| إرسال | `/send` | إرسال رسائل/ملفات |
| الرسائل | `/messages` | سجل الرسائل |
| مفاتيح API | `/api-keys` | إدارة المفاتيح |
| المستخدمون | `/users` | إدارة الفريق (admin) |
| سجل التدقيق | `/audit` | تتبع العمليات |
| الإعدادات | `/settings` | إعدادات النظام |

## تشغيل بيئة التطوير

```bash
PORT=23097 BASE_PATH=/whatsapp/ pnpm --filter @workspace/whatsapp-dashboard run dev
```

## ميزات واتساب المدعومة

- ✅ إرسال نص، صور، فيديو، صوت، ملفات، موقع، ملصقات
- ✅ QR Code عبر Socket.IO
- ✅ Webhook للرسائل الواردة
- ✅ ردود تلقائية
- ✅ جلسات متعددة بأدوار مختلفة

wppconnect يتطلب Chromium. في Replit sandbox هناك قيود. يعمل بالكامل على VPS أو Docker.
